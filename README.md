# mini_golf_tracker

A Flutter app for tracking mini golf players, courses, games, and scores.

## Local setup

Install Flutter outside of this repository and make sure `flutter` is available
on your shell `PATH`. Do not commit a Flutter SDK directory into this app repo.

```sh
flutter pub get
flutter analyze
flutter test
```
